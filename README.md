## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Usage

1. Paste your JSON in the input area (left panel)
2. Click "Format / Beautify" or press `Ctrl+Enter`
3. View the formatted result with syntax highlighting (right panel)
4. Copy the result with the "Copy" button
